# V9-Standard 快速入门

> **适用版本**：V9-Standard v2.9.0
> **前置要求**：WorkBuddy 已安装并可正常运行
> **最后更新**：2026-05-19

---

## 一、什么是 V9-Standard

V9-Standard 是 xmgl 项目管理治理框架的**标准推荐版**，包含 20 个核心技能，覆盖项目治理、记忆自保、提示词优化等日常所需全部能力。

**与 V9-Base / V9-Full 的差异**：
- V9-Base（5技能）：最小可行集，轻量入门
- **V9-Standard（20技能）**：日常推荐 ← 你在这里
- V9-Full（27技能）：完整治理体系，含数据库分析和批次调度

---

## 二、技能一览（20个）

| 记忆/会话 | 4 | context-survival, session-manager, window-aware, workbuddy-debug-helper |

## 三、部署步骤（零依赖）

### 第1步：告诉 AI 源路径

在 WorkBuddy 中输入（直接复制粘贴）：

```
请帮我把以下目录的 V9-Standard 技能文件部署到 ~/.workbuddy/skills/
源路径：{此处替换为你的 V9-Standard 目录绝对路径}
```

AI 会自动读取所有 20 个技能文件并安装到正确位置。

### 第2步：验证部署

重启 WorkBuddy，输入：

```
/skills
```

检查是否有 20 个技能显示在列表中。

### 第3步：开始使用

```
会话守护
```

激活 session-guard，然后开始项目追踪：

```
@project-tracker 记录事件
项目：myproj
事件：完成 V9-Standard 部署
标签：里程碑
```

## 四、版本选择建议

| 层级 | 技能数 | 适用场景 |
|:---|:--:|:---|
| V9-Base | 5 | 只需基本项目追踪，轻量使用 |
| V9-Standard | 20 | 日常项目管理（←推荐） |
| V9-Full | 27 | 需要数据库分析、批次调度等高级功能 |

---

## 五、升级路径

- 当前 V9-Standard 用户：可升级到 V9-Full（增加 7 个领域/批处理技能）
- V9-Base 用户：可直接部署 V9-Standard 覆盖/新增技能

升级方式相同：告诉 AI 新版本源路径即可。

## 六、每次会话的工作流

```
启动 WorkBuddy
    │
    ▼
会话守护（session-guard）  ← 检查状态，纠正漂移
    │
    ▼
项目追踪（project-tracker） ← 记录进展
    │
    ▼
任务门禁（checkpoint-guard）← 复杂任务断点
    │
    ▼
收工（shutdown-protocol）   ← 持久化变更到文件
```

## 七、常见问题

**Q：V9-Standard 够用吗？**
A：对绝大多数日常项目管理场景已足够。如需数据库变更分析、批次调度等高级功能，再升级 V9-Full。

**Q：部署后技能没有响应？**
A：需要重启 WorkBuddy 才能加载新技能。

**Q：可以和商业版同时用吗？**
A：可以。V9-Standard 是 Apache-2.0 许可，可自由使用和修改。
