## ⭐ 常用指令（3个）

### 01 project-tracker — 项目事件记录

| 项目 | 内容 |
|:---|:---|
| **作用** | 记录项目每日进展、生成周报、管理待办清单 |
| **触发词** | `记录事件` `今日进展` `周报` `汇报进度` |
| **安全边界** | 严格限定在 project.md 定义的项目范围内 |

**场景示例**：
```
记录事件 项目xmgl 今天完成了xxx
周报 项目xmgl
待办 xmgl
```

**约束**：事件日志超30条时自动归档前20条到 `_archive/`

---

### 02 shutdown-protocol — 收工协议

| 项目 | 内容 |
|:---|:---|
| **作用** | 会话结束前执行7步持久化：扫描变更→更新状态→写记忆→退出 |
| **触发词** | `收工` `shutdown` `wrap up` `结束会话` |

**场景示例**：
```
收工        # 日常收工，自动保存所有变更
shutdown    # 临时中断保存进度，下次恢复
```

**约束**：
- 🚫 禁止 `run_in_background`（BUG #07 冻结）
- 每次会话结束前必须执行

---

### 03 session-guard — 会话守护

| 项目 | 内容 |
|:---|:---|
| **作用** | 新会话启动时读取TRUTH-SOURCE快照，SHA256指纹校验，检测记忆漂移 |
| **触发词** | `会话守护` `检查记忆` `状态审计` `session guard` `drift check` |

**场景示例**：
```
会话守护    # 读取TRUTH-SOURCE对比当前记忆
检查记忆    # 检测上下文压缩导致的遗忘
```

**强制规则**：每次新会话启动后必须调用

### 05 checkpoint-guard — 任务门禁

| 项目 | 内容 |
|:---|:---|
| **作用** | 复杂任务（>5步）强制门禁检查，支持跨会话续传 |
| **触发词** | `任务门禁` `checkpoint` `断点续传` `复杂任务` |

**场景示例**：
```
我要执行数据库迁移...   # 自动触发门禁
断点续传               # 恢复上次checkpoint继续执行
```

**触发条件**：预估步骤>5步 / 用户说"复杂任务" / 跨会话执行
