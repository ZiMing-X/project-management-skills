## 版本定位

V9-Base 是 xmgl 治理框架的**最小可行集**。仅包含 5 个核心技能，覆盖项目生命周期最基本的骨架：会话守护 → 项目追踪 → 任务门禁 → 收工持久化。

**核心差异（vs V9-Standard / V9-Full）**：
- V9-Base：5技能，项目生命周期骨架
- V9-Standard：20技能，日常推荐工具集
- V9-Full：27技能，完整治理体系

**适用场景**：
- 只需基本项目追踪的用户
- 轻量级会话管理
- 作为后续升级到 Standard/Full 的起点

---

## 三层架构

```
V9-Full (27)
  ├── V9-Standard (20) → 日常推荐
  └── V9-Base (5)      → 最小可行集 ← 你在这里
```

### Base 层（5技能）— 项目生命周期骨架

| # | 技能 | 定位 |
|:--|:---|:---|
| 1 | project-tracker | 事件日志/周报/待办 |
| 2 | project-manager | 项目文档体系管理 |
| 3 | shutdown-protocol | 收工持久化协议 |
| 4 | session-guard | 会话启动守护 |
| 5 | checkpoint-guard | 复杂任务门禁 |

### 升级路径

| 层级 | 技能数 | 新增能力 |
|:---|:--:|:---|
| V9-Base | 5 | 核心治理骨架 |
